# Medications App & API

Este projeto é um ecossistema completo para listagem de medicamentos. Esse repositório é o código da aplicação mobile e visual. Para ver o backend vá para o repositório disponibilizado no link abaixo.

**[Backend (API)](https://github.com/cldavii/medicationsapi)**

---
## 🛠️ Tecnologias Comuns Recomendadas

- **Runtime/Linguagem:** TypeScript e Expo
- **Gerenciador de Pacotes:** npm
---
## 🚀 Como Configurar e Executar Localmente
Para executar o código localmente, você deve seguir alguns passos:
1. Clone do repositório
```
git clone https://github.com/cldavii/medicationsapi.git
```
2. Instalação das denpendências
```
npm i
```
3. Inserir o IP da máquina que está rodando o projeto
```
/src/services/api.ts - Linha 2
```
4. Execute o backend

[Como executar o backend?](https://github.com/cldavii/medicationsapi)
